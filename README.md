## Package Build Android for ArchLinux
 
 #### Packages

Turn on multilib

sudo nano /etc/pacman.conf

[multilib]

Include = /etc/pacman.d/mirrorlist

#### How to install 
```
git clone
https://github.com/ChoungMyung/ArchLinuxAndroidBuild.git
```
```
cd /ArchLinuxAndroidBuild/
```
```
chmod +x install.sh
```
```
./install.sh
```
